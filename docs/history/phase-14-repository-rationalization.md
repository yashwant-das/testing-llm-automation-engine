## Decisions Made

1. **`src/agents/generator.py` stays** — `generate_test_script()` is actively called by `generation_service.py` and `benchmarks/generation/runner.py`. Only the dead `run_generated_test()` function is removed.

2. **`src/agents/healer.py` stays** — It has an active `if __name__ == "__main__":` CLI entrypoint that is documented. The stale phase comment is removed.

3. **`src/agents/vision.py` deleted entirely** — `analyze_visual_ui()` has no callers. `vision_service.py` is the active implementation. `encode_image()` is only called by the dead function.

4. **`_extract_code_block` and `_extract_json_block` promoted to public** — Both are called by production code (`generator.py`, `vision_service.py`, `parse_llm_response()`). The leading underscore signals "private" but these are stable, well-tested utilities. Promoting removes the inconsistency between usage and naming.

5. **`benchmarks/mutations/` stays** — Actively tested. `unit_test_evaluation.py` imports from it and tests all four mutation types.

6. **`benchmarks/intent_validation/fixtures/` stays** — Empty fixture set is intentional gap (R-003 backlog item). The infrastructure is ready.

7. **`docs/technical-debt.md` archived** — All 15 items resolved. Document is historical. Future debt tracks in `docs/backlog.md` "Cleanup Work" section.

8. **`progress/` directory deleted after archiving** — No ongoing phases warrant keeping it.
